# SkillConnect

## Short setup & build guide

### Requirements
- Node.js 18+
- npm 9+

### Install
1) `npm install`
2) `npm run init-db`

### Run (dev)
- Backend: `npm run dev`
- Frontend: `npm run start:frontend`

### Build (production)
1) `npm run build:frontend`
2) `npm run dev`

After start:
- Frontend: http://localhost:4200
- API: http://localhost:3000/api
